        # 使用教程

        提示

        会员服务可以帮助我们更好的提供服务，同时也可以获得更多的服务，请根据需求自行选择。

        导航站址：https://vlink.cc/jinglin (opens new window)

        独享授权码独特优点：

        • 授权码具有专属性；
        • 聊天记录数据具有强私密性，只要授权码不变，聊天数据会存在的，但重要数据请做好备份;
        • 不易人数多受官网次数限制，随时可用；
        • 授权码在有效期内不会失效；
        • 文件上传、数据分析、新版绘画、图片识别、GPTs商店、高级语音对话、代码写作画布、项目空间、搜索引擎等全部高级功能。

        仅三人共享授权码：

        • 授权码具有专属性；
        • 聊天记录数据仅共享三人可见，重要数据请及时做好备份，以防授权码过期聊天记录不在了，或者被共享人误删聊天记录，不可控;
        • 每个共享授权码只能自己用，否则容易受授权码次数限制；
        • 授权码在有效期内不会失效；
        • 文件上传、数据分析、新版绘画、图片识别、GPTs商店、高级语音对话、代码写作画布、项目空间、搜索引擎等全部高级功能。

        独享和共享授权码区别：

        • 独享授权码次数是自己独用的，共享授权码是系统已自动随机分配三人累计合用次数的（一个共享授权码只是三人中的一员），每个授权码都要注重保护，每个模型次数是一定的；
        • 独享授权码私密性更强，聊天记录只有自己一人能知晓看到；共享授权码聊天记录是仅用此码的三人相互可见；只要授权码不变，聊天数据会存在的，但重要数据请做好备份，尽量复制到本地，养成备份习惯起到双保险；
        • 独享授权码续费可以保持原授权码不变，续费之前的聊天记录会存在的；共享授权码续费是需要重新开通新授权码的，续费之前的记录会看不到的，建议及时保存备份。聊天记录是随着授权码独立保存的，不同授权码之间聊天记录不可见；
        • 独享授权码是可以自定义更改的，但更改之前的聊天记录会看不到的；共享授权码是更改不了的。

        # 遇到问题

        • 重要聊天记录尽量复制备份到本地，避免因网络波动个别丢失，养成备份习惯起到双保险
