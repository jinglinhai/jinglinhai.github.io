        # 使用教程

        提示

        会员服务可以帮助我们更好的提供服务，同时也可以获得更多的服务，请根据需求自行选择。 独享/仅三人共享PLUS授权码独特优点：

        • 授权码具有专属性；
        • 聊天记录数据具有私密性，只要授权码不变，聊天数据会存在的，但重要数据请做好备份;
        • 不易人数多受官网次数限制，随时可用；
        • 授权码在有效期内不会失效；
        • 文件上传、数据分析、DALLE绘画、图片识别、GPTs商店、语音对话等全部功能；
        • 初次使用请仔细阅读ChatGPT官方版使用教程 (opens new window)。
        • 网站教程: GPT相关教程也可看仓库，也会不定时持续更新教程和分享优秀的gpts工具地址 gitee仓库 (opens new window) github仓库 (opens new window) 全站点网站首页也有相关教程地址！

        # 使用方法

        • 购买授权码：https://store.jlkjy.cn (opens new window)
        • 使用入口填写购买后得到的授权码: https://cloud.jlkjy.cn (opens new window)
        • 一个授权码可随意切换所有官方账号使用
        • 独享授权码可联系客服更改授权码，并且可保持原授权码不变来续费

        # 付费服务

        购买会员后，您将接收到会员专属授权码，请妥善保管。授权码是您的会员身份凭证，请勿泄露。点击查看详细使用教程 (opens new window)，进会员站点即可使用。其他具体详情去看看

        # 遇到问题

        • 提示4.0/4o没次数：换车并新建对话
        • 重要聊天记录尽量复制备份到本地，避免个别丢失
        • 若授权码忘记或者找不到，也可通过点击访问授权码找回 (opens new window)查询订单号或联系方式找回专属授权码
        • 上述无法解决或者其他遇到的问题，请联系客服（微信客服或售后Q群群主）
